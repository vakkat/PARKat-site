Privacy Policy – PARKat

Effective date: 18 February 2026

1. Data We Collect

  • Precise location data (GPS)
  • Sensor data (accelerometer)
  • Trip timestamps
  • Anonymous user identifier (Firebase UID)

2. Purpose of Data Collection

The PARKat application collects data exclusively for academic research purposes. The objective is to analyze parking search behavior and urban mobility patterns.

3. Data Storage and Security

All data is securely transmitted and stored using Firebase Firestore. Communication is encrypted using HTTPS protocols.

4. Data Retention

Data is retained only for the duration of the research project. Aggregated or anonymized data may be used for scientific publication.

5. User Rights

Users may request deletion of their data at any time by contacting:
your-email@ntua.gr

6. Research Institution

This research is conducted by the National Technical University of Athens (NTUA).
