PARKat Privacy Policy

Effective date: 12/02/2026

1. Data We Collect

  • Precise location data (GPS)
  • Sensor data (accelerometer)
  • Trip timestamps
  • Anonymous user identifier (Firebase UID)

2. Purpose of Data Collection

Data is collected for academic research purposes to analyze parking search behavior and urban mobility patterns.

3. Data Storage

Data is securely stored in Firebase Firestore using encrypted communication.

4. Data Retention

Data is retained only for the duration of the research project and may be anonymized for scientific publication.

5. User Rights

Users may request deletion of their data by contacting: kvakrinou@mail.ntua.gr

6. Contact
